En este Curso conocerás la forma en la que se representan y realizan operaciones con datos computaciones, conociendo el sistema binario, ocal y hexadecimal.

Contenido

La representación de datos dentro de la computadora se realiza mediante patrones de bits, sin embargo existen otras notaciones que son utilizadas para facilitar la manipulación de grandes secuencias de bits, estos son los llamados sistemas numéricos decimal, hexadecimal y octal. En esta sección conocerás dichos sistemas y la forma de convertir las grandes secuencias de bits a éstos

 

Sistema binario

El Sistema Binario está conformado solamente por dos dígitos: 0 y 1 (bits), los cuales representan impulsos eléctricos dentro de la computadora que le indican "apagado" o "encendido", "falso" o "verdadero". Una computadora sólo puede entender este lenguaje y por eso se le conoce como lenguaje máquina

Contenido

 

Haz clic en cada una de las secciones del recurso para que conozcas las características generales del sistema binario.

  • Características:

    •Su base es el número 2.

    •Se compone de 2 elementos que son: 0 y 1.

    •Un número binario tiene dos elementos importantes, posición y exponente. Los cuales son explicados con el siguiente ejemplo:

    Se tiene el número binario 01001110.

    •Posición: lugar que ocupan de derecha a izquierda comenzando por 0.

    •Exponente de base 2: es la posición de cada dígito binario.

    01001110
    Posición:76543210
    Exponente base 2:2726252423222120

  • Tomando en cuenta la información anterior sobre las posiciones de cada bit, la conversión del sistema binario al decimal se realiza como sigue:

    Convertir 0101101 a decimal

    1. Elevar la base 2 a un exponente que sea igual a la posición de cada dígito binario, pero sólo se eleva la potencia de aquellos bits iguales a 1. Los bits iguales a 0, no tomarlos en cuenta.

    0101101
    Posición:6543210
    Exponente base 2:26252423222120

    cuenta    		84No
    cuenta    		1
